## ✨ Coursefit 소개 ✨

📍Coursefit은 <em>course</em>와 <em>fit</em>을 합쳐 <em>나에게 딱 맞는 강의</em>라는 메시지를 담고 있는 서비스명입니다

Coursefit은 사용자가 자신이 수강한 과목과 그 성적에 기반하여 다음으로 들을 강의를 추천 받을 수 있는 서비스입니다. 사용자가 추천받고 싶은 강의 종류(전공/교양)를 선택하면 강의계획서와 실수강생들의 강의평 데이터를 통합 분석한 LLM 결과를 기반으로 사용자가 높은 성취감을 얻을 수 있는 강의와 그 추천 이유를 함께 제공합니다.

<br />

## 🪄 주요 기능

- 성적표 이미지 업로드 및 과목/성적 자동 인식📄
- 전공/교양 강의 중 추천받고 싶은 유형 선택✔️
- 강의계획서, 강의평 기반 개개인의 성적과 분야에 맞는 강의 추천 및 추천 이유 제공📝
- 분석 결과 히스토리 관리📚
