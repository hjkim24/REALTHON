[강의요목]
본 과목은 데이터베이스의 기본 개념을 관계형 데이터 모델, 관계형 대수 및 SQL을 통해 설명하고, 나아가 DBMS의 필수구성요소들인 저장장치, 색인시스템, 질의 처리 및 최적화, 트랜잭션 관리 등을 소개한다.

[주차별 강의 내용]
1주: Course Introduction
2주: Basic Database Concept (Assignment #0)
3주: Relational Model, Relational Algebra
4주: Basic SQL
5-6주: Advanced SQL (Assignment #1) / Database design using E-R Model
7주: Relational Database Design
8주: Midterm Exam (중간고사)
9주: File/Table Organization, Storage System
10주: Transaction
11주: Concurrency Control (Assignment #2)
12주: Indexing
13주: Logging & Recovery
14주: Query Processing 1 (Assignment #3)
15주: Query Processing 2
16주: Final Exam (기말고사)
